Aptana: advanced IDE for Web 2.0

Wednesday, 26 July 2006

linux ⋅ javascript ⋅ java ⋅ windows ⋅ macosx ⋅ sw ⋅ it ⋅ opensource ⋅ english ⋅ devtools ⋅ projects

Aptana is a robust, JavaScript-focused IDE for building dynamic web applications. Highlights include the following features:

* Code Assist on JavaScript, HTML, and CSS languages, including your own JavaScript functions
* Outliner that gives a snapshot view of your JavaScript, HTML, and CSS code structure
* Error and warning notification for your code
* Support for Aptana UI customization and extensions
* Cross-platform support
* Free and open source. (Source available soon)

Aptana seems to be a very powerfull IDE for Javascript (finally!!!). The biggest features is the Integrated documentation, merged with a "suggestions-system" that indicate standard and non standard tags and attributes (indicating the browser compatibility with icons).
