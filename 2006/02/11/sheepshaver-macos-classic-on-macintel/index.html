SheepShaver: MacOS Classic on MacIntel

  • Saturday, 11 Feb 2006
  • english • sw • it • projects • opensource
  • 125 words

This application was bord with the run Classic Application in any operating system. It seems to be an “Emulation-Layer” (so, not a “Virtuallization-Layer”) and… it’s for available (and supported) on various systems:

  • Linux (i386, ppc, x86_64)
  • MacOS X (i386, ppc)
  • Darwin
  • NetBSD 2.0
  • FreeBSD 5.3
  • Windows for x86

As you probably know, Apple, with new MacIntel, has cutted-off the support for Classic: it was a virtuallization, not an emulation and then… a simple recompilation it’s not enough.

As you can read before, the project now support MacOSX (i386) machine: reading on the official site of SheepShaver it’s clear that the support is experimental. But it’s really ;) .

It will can be the simplest solution for users that use MacOS application right now. Source, Melablog.
