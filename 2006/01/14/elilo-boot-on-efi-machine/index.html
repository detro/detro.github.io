elilo: boot on EFI machine

Saturday, 14 Jan 2006 — english sw it hw opensource linux projects

Linux, as we can expect from the linux-comunity, are ready for the EFI system that will substitute the old BIOS system on our (x86) machine.

EFI is the boot-system used on the new MacIntel machine, presented from Apple at Jan 10. What it means? It means that we can install and run on EFI-enabled machine every Operating System, also which that does not "understand" EFI (WinXP is the biggest example).
