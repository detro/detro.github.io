Firefox, Thunderbird and Camino: first Universal Binary build

16 Jan 2006

Developers of Firefox, Thunderbird and Camino are working on a Universal Binary release: they have promised a official release in march.

By now, here are sufficiently stable releases. But, problems still remain: Java-plugin and Flash-plugin does not run, right now. On MacIntel (if you are so lucky ;) ) use Firefox build for PPC over Rosetta.

Source, Melablog.
