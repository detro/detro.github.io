## Requirements

* Ruby (2.4.x WITH dev tools)
* Python (2.7.x)
* NVM (1.1.x)

## Dependencies

### Ruby

```
gem install bundler jekyll
cd <blog source directory>
bundle install
```

### NVM

```
nvm list available
# choose latest LTS
nvm install <latest LTS>
nvm use <latest LTS>
npm install less -g
```
