Linux, as we can expect from the linux-comunity, are ready for the [EFI system](http://en.wikipedia.org/wiki/Extensible_Firmware_Interface) that will substitute the old [BIOS system](http://en.wikipedia.org/wiki/BIOS) on our (x86) machine.

EFI is the boot-system used on the [new MacIntel machine](http://www.detronizator.org/2006/01/12/letter-to-apple-support/), [presented from Apple at Jan 10](http://www.detronizator.org/2006/01/12/its-too-late/). What it means? It means that we can install and run on EFI-enabled machine every Operating System, also which that does not "understand" EFI (WinXP is the biggest example).

[Here](http://elilo.sourceforge.net/cgi-bin/blosxom) the home page of [elilo](http://elilo.sourceforge.net/cgi-bin/blosxom), <em>the EFI linux boot loader</em>.
