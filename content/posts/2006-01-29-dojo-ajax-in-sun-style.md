> [Dojo](http://dojotoolkit.org/) is the Open Source JavaScript toolkit that helps you build serious applications in less time. It fills in the gaps where JavaScript and browsers don't go quite far enough, and gives you powerful, portable, lightweight, and tested tools for constructing dynamic interfaces.



This project has Sun behind itself: on [Downloadblog](http://www.downloadblog.it/post/1285/un-video-su-ajax-dagli-sviluppatori-sun) there is a post dedicated to a [Videocast](http://developers.sun.com/channel/01_06/index.jsp?video=9d26a212-2514-4892-a546-b98bdab41a69) of [Sun Developers](http://developers.sun.com/) about Dojo and [Java Studio Creator](http://www.ossblog.it/post/50/sun-java-studio-creattor-e-studio-enterprise-8-gratuiti).

The "packages" organization of Dojo is very similar to Java Library: much coherent and clean.
