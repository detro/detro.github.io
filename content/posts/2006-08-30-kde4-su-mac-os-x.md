![Konqueror on MacOSX](http://ranger.befunk.com/gallery2/d/9242-2/konqueror-20060826.png)

It can be REAL!
After the sufficient, but instable, KDE3 compiled on Mac OSX, [seems](http://www.racoonfink.com/archives/000700.html) that [KDE4's developers are working to run it on Mac OS X](http://ranger.users.finkproject.org/kde/).

This is possible because [Trolltech](http://www.trolltech.com/) have released many months ago [QT4](http://www.trolltech.com/products/qt) [for Mac OS X](http://www.trolltech.com/products/qt/features/crossplatform/mac): a very powerfull, smart and fast C++ Framework... integrated with [Aqua](http://www.apple.com/macosx/overview/aquauserinterface.html).

<!--more-->
