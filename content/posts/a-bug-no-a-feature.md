... a Windows "[feature](http://www.securityfocus.com/brief/91)" created in the 1990, in win 3.0, is <strong>STILL ALIVE!</strong> 8O

The problem? This "feature", the function [SetAbortProc](http://msdn.microsoft.com/library/default.asp?url=/library/en-us/gdi/prntspol_0883.asp), is used by windows when a print-job must be stopped.

1990! We are in 2006! We (YOU) are on Windows XP! 8O
