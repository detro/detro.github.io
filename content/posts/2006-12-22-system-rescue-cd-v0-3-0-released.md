Main changes:


>> <strong>0.3.0) 2006-12-20:</strong>
>> * Updated the kernel to Linux-2.6.18.5 with Reiser4
>> * Added xorg graphical environment with WindowMaker
>> * <strong>Added gparted-0.3.3 to replace qtparted</strong>
>> * <strong>Added ntfs3g-beta-20061212 (full NTFS Read/Write)</strong>
>> * Added graphical editors (gvim, leafpad)
>> * Added graphical web browsers (firefox-2.0 and dillo)
>> * Updated the file-system tools (parted, e2fsprogs, xfsprogs, ...)
>> * Updated the system (glibc, clamav, ...)
>> * Updated the bootdisks 
>



I used it a couple of times to fix my linux-box and, in general, my pc. It's the <em>swift knife</em> of every good admin. The only "fault" of it? The project is growing only on x86: PPC is sucking.

p.s. I want to highlight to you the most important features in this release, for me: the usage of [GParted](http://gparted.sourceforge.net/), better of any other partitioning tool on linux, and the driver [NTFS3G](http://www.ntfs-3g.org/).
